# Changelog

All notable changes to this project will be documented in this file.

The format is based on
[Keep a Changelog](https://keepachangelog.com/).

## [0.1.0] — 2026-03-08

Initial release as a standalone repository. The registry was previously
part of [labeille](https://github.com/devdanzin/labeille).

### Added
- 5,000+ package YAML files with installation and testing instructions.
- ~1,800 enriched packages with working install and test commands.
- `index.yaml` summary index sorted by PyPI download count.
- `schema.yaml` for version compatibility checking with labeille.
- `migrations.log` tracking applied schema transformations.
- README with complete field schema, enrichment guide, and CLI reference.
- CONTRIBUTING.md with enrichment workflow and YAML conventions.
- CLAUDE.md for AI-assisted enrichment.
